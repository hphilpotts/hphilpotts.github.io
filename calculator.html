Simple Static Calculator - Mini Project (in progress)


Deployment | GitHub | README

This is a simple static site 'old school' calculator mini project, using HTML5, CSS and 'vanilla' JS. I chose this mini project as an opportunity to revisit some of the first skills I learned when embarking on my coding journey. Recognising the risk of skills atrophy, I decided to go with one of the 'classic' mini projects in order to revisit these key skills.


I've found this to be a really interesting side-project to take on; not least because it ended up being far more involved than I initially thought, especially when attempting to replicate the functionality and typical user inputs of a 'classic' calculator without the use of a library. Equally, it has been refreshing to go back to using simple CSS (and quite a lot of Googling) rather than heading straight to a library like Materialize.

My focus has been on writing clean, readable code whilst working within the technical limitations of the project as a simple static site. Looking back at the code I used in my GA Project 1 (Simpsons' Guess Who?) - which mirrors this project in terms of technologies used (or rather, not used) - I can see a striking difference in structuring and readability!

This is still very much a work in progress! Its relative simplicity has meant that I can 'chip away' at improvements in small bursts of time in between work, applications, further learning and so on. As well as working through my list of improvements / bug fixes, my ambition is to convert this to TypeScript in order to build on my learning. I very quickly realised how much this project would benefit from the use of TS and am excited to see how the transformation goes!
